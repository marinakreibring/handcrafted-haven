# Handcrafted Haven

Handcrafted Haven is a web application marketplace for artisans and crafters to showcase and sell handmade products.

## Technologies
- Next.js
- Node.js
- MongoDB
- Tailwind CSS
- Vercel

## Features
- Seller profiles
- Product listings
- Reviews and ratings
- Responsive design
- Accessibility support
